What are CSS Custom properties (also called CSS Variables)?

They are values that can be reused throughout your styles and defined once. When declaring your custom property, they always start with two dashes and are assigned a value (e.g. --main-color). This will also need to be used with the var() function. To store your values, it is common practice to define the variables on :root for easy management of colours, fonts, and spacing across your whole website.

Why does it matter?

Modernized CSS

Instead of relying on preprocessors like Sass, Custom Properties are built into CSS itself - faster, simpler and more powerful.

More simplicity with the way we write css

With Custom properties, developers have the freedom to create their own reusable values - CSS couldn't do something like this before.

Solves a Universal Problem

When custom properties weren't around, stylesheets were filled with random and repeated values. With Custom Properties, it makes every developer's life easier.

More Scalable

Building big websites becomes easier to update and organized with the simplified and consistent rules that custom properties bring.

They help developers work together

This matters because, as developers build big websites or projects, they can share the same properties/variables for colours, fonts and spacing. This promotes consistency in everyone's design while still allowing room for individual creativity.

Less errors and mistakes

Instead of retyping the values, reusing them helps prevent errors and broken designs.
